# export-dashboard.html

**Location:** `THE WARATAH/SHIFT REPORT SCRIPTS/export-dashboard.html`
**Type:** Bundled React sidebar application
**Opened via:** Menu > Daily Reports > Open Export Dashboard (no password required)

---

## What This File Does

This is a pre-built React application bundled into a single HTML file. It provides an interactive export dashboard that opens as a sidebar inside the Google Sheets spreadsheet, giving managers a visual interface for managing shift report exports.

Like `analytics-viewer.html`, this file contains minified React, Tailwind CSS, and Lucide icons all inlined — no external dependencies are loaded at runtime.

---

## How It's Used

1. A user opens the Waratah shift report spreadsheet
2. They go to **Waratah Tools > Daily Reports > Open Export Dashboard**
3. `UIServer.js` calls `HtmlService.createHtmlOutputFromFile('export-dashboard')` and opens it as a sidebar
4. The React app renders inside the sidebar panel
