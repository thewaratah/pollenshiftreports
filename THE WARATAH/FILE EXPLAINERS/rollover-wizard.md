# rollover-wizard.html

**Location:** `THE WARATAH/SHIFT REPORT SCRIPTS/rollover-wizard.html`
**Type:** Bundled React modal dialog
**Opened via:** Menu > Admin Tools > Weekly Reports > Weekly Rollover (In-Place) > Run Rollover Now

---

## What This File Does

This is a pre-built React application bundled into a single HTML file. It provides an interactive wizard for the weekly rollover process, giving admins a preview of what will be cleared and a confirmation step before executing.

Like `analytics-viewer.html` and `export-dashboard.html`, this file contains minified React, Tailwind CSS, and Lucide icons all inlined — no external dependencies are loaded at runtime.

---

## How It's Used

1. An admin opens the Waratah shift report spreadsheet
2. They go to **Waratah Tools > Admin Tools > Weekly Reports > Weekly Rollover (In-Place) > Run Rollover Now**
3. They enter the admin password
4. `UIServer.js` calls `HtmlService.createHtmlOutputFromFile('rollover-wizard')` and opens it as a 500x640 modal dialog
5. The React app renders inside the dialog with preview data and confirmation controls

## Server-Side Functions It Calls

The wizard communicates with the server via `google.script.run`:

| Call | What It Does |
|------|-------------|
| `getRolloverPreview()` | Fetches a dry-run summary of what the rollover will do (data to be cleared, dates to be updated) |
| `executeRollover()` | Triggers the actual `performWeeklyRollover()` function |

---

## Important Notes

- **Do not edit this file by hand.** It's a build artifact from a separate React project.
- **Password-gated** — only accessible through the Admin Tools submenu, which requires the admin password.
- The wizard provides a safer rollover experience than running the function directly — users can preview changes before committing.
